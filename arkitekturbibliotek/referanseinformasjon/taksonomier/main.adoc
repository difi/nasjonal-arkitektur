== Hvorfor taksonomier?

* Navigering via taksonomi/kategorier

* Søkeoptimalisering

* Flerspråklige beskrivelser 

* Menystruktur for offentlige tjeneste

* Benchmarking - hvike tjenester/prosesser/leverandøer gir best tjeneste (ytelse, pris)

* Capability lookup  

* Fjerne redundans/overlapp 

* Porteføljestyring – proaktivt, behovsdrevet

* Interoperabilitet  - interoperable prosesser, kontroll på avhengigheter
